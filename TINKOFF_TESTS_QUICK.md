# 🧪 Быстрая инструкция: Тесты Тинькофф #7 и #8

## ✅ Тест #7: Формирование чека

1. Зайти на https://asia-sib.web.app
2. Создать заказ
3. Оплатить картой: **4000 0000 0000 0101** (срок: 12/30, CVV: 111)
4. Проверить в ЛК Тинькофф → статус "Подтвержден"

---

## ❌ Тест #8: Отмена платежа

1. Зайти на https://asia-sib.web.app
2. Создать заказ
3. Оплатить картой: **5000 0000 0000 0108** (срок: 12/30, CVV: 111)
4. Открыть https://asia-sib.web.app/test-cancel
5. Нажать "Отменить платеж"
6. Проверить в ЛК Тинькофф → статус "Возвращен полностью"
