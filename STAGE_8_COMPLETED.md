## Как использовать

### Создание совместного заказа:
1. Добавьте товары в корзину
2. Выберите тип доставки "Совместный заказ с соседями"
3. Выберите дату и время доставки
4. Укажите адрес доставки
5. Нажмите "Создать совместный заказ"
6. Поделитесь кодом/ссылкой/QR-кодом с соседями

### Присоединение к заказу:

**Способ 1: По коду**
1. Нажмите "Присоединиться" в шапке сайта
2. Введите код заказа (SOSEDI-XXXX)
3. Нажмите "Присоединиться"

**Способ 2: По ссылке**
1. Перейдите по ссылке от организатора
2. Автоматически откроется страница заказа

**Способ 3: По QR-коду**
1. Отсканируйте QR-код камерой телефона
2. Перейдите по ссылке

### Оплата:
1. Сначала оплатите товары (кнопка "Оплатить товары")
2. После закрытия пула менеджер рассчитает доставку
3. Оплатите свою долю доставки (кнопка "Оплатить доставку")

### Выход из заказа:
- Можно выйти только ДО оплаты товаров
- После оплаты выход невозможен

## Технические детали

### Формат кода группового заказа:
```
SOSEDI-XXXX
```
где XXXX - 4 случайных буквенно-цифровых символа

### Структура GroupOrder:
```typescript
{
  id: string;
  code: string;
  organizerId: string;
  participants: Array<{
    userId: string;
    userName: string;
    items: CartItem[];
    total: number;
    productsPaid: boolean;
    deliveryPaid: boolean;
  }>;
  address: string;
  deliveryDate: string;
  deliveryTimeSlot: string;
  deliveryPoolId: string;
  status: 'open' | 'closed';
  deliveryPrice?: number;
  deliveryShares?: Record<string, number>;
}
```
