# Автоматический расчет доставки

## Что изменилось

Теперь стоимость доставки рассчитывается **автоматически** через API Яндекс Доставки, когда продавец подтверждает заказ. Менеджер больше не нужен для добавления стоимости доставки.

## Процесс заказа

1. **Покупатель** оформляет заказ
2. **Продавец** подтверждает заказ
3. **Система автоматически**:
   - Рассчитывает стоимость доставки через Яндекс API
   - Добавляет её к заказу
   - Отправляет заказ на оплату покупателю
4. **Покупатель** оплачивает заказ с доставкой
5. **Продавец** собирает заказ
6. **Менеджер** отправляет в доставку

## Настройка

### 1. Получите токены

- **Яндекс Доставка**: https://b2b.taxi.yandex.ru/
- **Яндекс Геокодер**: https://developer.tech.yandex.ru/services/

### 2. Добавьте в `.env` файл

```env
# Yandex Delivery API
REACT_APP_YANDEX_DELIVERY_TOKEN=ваш_токен_яндекс_доставки
REACT_APP_YANDEX_GEOCODER_KEY=ваш_ключ_геокодера
REACT_APP_WAREHOUSE_ADDRESS=Москва, ул. Примерная, 1
```

### 3. Укажите адрес склада

Замените `REACT_APP_WAREHOUSE_ADDRESS` на реальный адрес вашего склада/павильона.

## Резервный вариант

Если API недоступен или произошла ошибка:
- Система автоматически установит фиксированную стоимость доставки **300₽**
- Заказ продолжит обработку в обычном режиме

## Файлы

- `src/utils/yandexDelivery.ts` - утилита для расчета доставки
- `src/App.tsx` - автоматический расчет при подтверждении заказа
- `src/pages/ManagerDashboard.tsx` - кнопка "Авто" для ручного расчета (опционально)

## Преимущества

✅ Не нужен менеджер для каждого заказа  
✅ Точная стоимость доставки в реальном времени  
✅ Автоматизация процесса  
✅ Резервная фиксированная цена при ошибках
