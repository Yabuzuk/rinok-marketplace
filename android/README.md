# Ринок - Android WebView App

## Сборка APK

1. Установите Android Studio
2. Откройте папку `android` как проект
3. Добавьте иконки в папки `res/mipmap-*`
4. Выполните сборку: Build → Build Bundle(s) / APK(s) → Build APK(s)

## Автоматические обновления

Приложение автоматически отображает изменения сайта https://asia-sib.web.app без необходимости обновления APK.

## Возможности

- Полноэкранный WebView
- Поддержка JavaScript
- Локальное хранилище
- Кнопка "Назад" для навигации
- Автоматическая загрузка обновлений сайта
