# Пошаговая инструкция: Настройка автоматического подтверждения оплаты

## ШАГ 1: Получить ключ Firebase Admin SDK

### 1.1 Открыть Firebase Console
Перейти по ссылке: https://console.firebase.google.com/project/asia-sib/settings/serviceaccounts/adminsdk

### 1.2 Скачать ключ
- Найти кнопку **"Generate new private key"** (желтая кнопка)
- Нажать на неё
- В появившемся окне нажать **"Generate key"**
- Файл `asia-sib-firebase-adminsdk-xxxxx.json` скачается в папку "Загрузки"

### 1.3 Открыть файл
- Найти скачанный файл в папке "Загрузки"
- Открыть его в Блокноте или любом текстовом редакторе
- Оставить открытым - данные из него понадобятся дальше

## ШАГ 2: Добавить переменные в Vercel

### 2.1 Открыть Vercel
- Перейти на https://vercel.com/
- Войти в аккаунт
- Найти проект **"rinok"** в списке проектов
- Нажать на него

### 2.2 Открыть настройки
- В верхнем меню нажать **"Settings"**
- В левом меню выбрать **"Environment Variables"**

### 2.3 Добавить первую переменную
- Нажать кнопку **"Add New"** или **"Add Variable"**
- В поле **"Key"** (или "Name") написать: `FIREBASE_PROJECT_ID`
- В поле **"Value"** написать: `asia-sib`
- Поставить галочки: ✅ Production, ✅ Preview, ✅ Development
- Нажать **"Save"**

### 2.4 Добавить вторую переменную
- Снова нажать **"Add New"**
- В поле **"Key"** написать: `FIREBASE_CLIENT_EMAIL`
- В поле **"Value"** скопировать из JSON файла значение `"client_email"` 
  (будет примерно: `firebase-adminsdk-xxxxx@asia-sib.iam.gserviceaccount.com`)
- Поставить галочки: ✅ Production, ✅ Preview, ✅ Development
- Нажать **"Save"**

### 2.5 Добавить третью переменную
- Снова нажать **"Add New"**
- В поле **"Key"** написать: `FIREBASE_PRIVATE_KEY`
- В поле **"Value"** скопировать из JSON файла значение `"private_key"` 
  (весь текст от `-----BEGIN PRIVATE KEY-----` до `-----END PRIVATE KEY-----\n`)
  ВАЖНО: копировать со всеми `\n` как есть в файле!
- Поставить галочки: ✅ Production, ✅ Preview, ✅ Development
- Нажать **"Save"**

## ШАГ 3: Задеплоить на Vercel

### 3.1 Открыть командную строку
- Нажать Win+R
- Написать `cmd` и нажать Enter

### 3.2 Перейти в папку проекта
```
cd e:\rinok
```

### 3.3 Задеплоить
```
vercel --prod
```

Если команда не найдена, сначала установить:
```
npm i -g vercel
vercel login
vercel --prod
```

### 3.4 Дождаться завершения
- Процесс займет 1-2 минуты
- В конце появится ссылка типа: `https://rinok.vercel.app`

---

## ШАГ 4: Настроить webhook в Тинькофф

### 4.1 Открыть личный кабинет Тинькофф Бизнес
- Перейти на https://business.tinkoff.ru/
- Войти в аккаунт

### 4.2 Найти настройки эквайринга
- В меню найти раздел **"Эквайринг"** или **"Интернет-эквайринг"**
- Найти терминал с номером **1771142972317**
- Нажать **"Настройки"** или **"Редактировать"**

### 4.3 Указать URL webhook
- Найти поле **"URL для уведомлений"** или **"Notification URL"**
- Вставить: `https://rinok.vercel.app/api/tinkoff-webhook`
- Нажать **"Сохранить"**

---

## ШАГ 5: Проверка

### 5.1 Создать тестовый заказ
- Зайти на https://asia-sib.web.app
- Создать заказ
- Нажать "Оплатить"

### 5.2 Оплатить
- Оплатить любым способом (карта/СБП)

### 5.3 Проверить статус
- Вернуться на сайт
- Статус заказа должен автоматически измениться на "Оплачен"

### 5.4 Посмотреть логи (если что-то не работает)
- Зайти на vercel.com → проект rinok
- Deployments → последний деплой → Functions → Logs
- Там будут видны все запросы от Тинькофф

---

## Готово! 🎉

Теперь при каждой оплате:
1. Пользователь оплачивает → Тинькофф обрабатывает
2. Тинькофф отправляет уведомление на ваш webhook
3. Webhook автоматически меняет статус заказа на "paid"
4. Пользователь видит обновленный статус

Работает даже если пользователь закрыл страницу!
