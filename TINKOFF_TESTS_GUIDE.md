# Прохождение тестов Тинькофф №7 и №8

## Подготовка

1. **Временно заменить файл для тестов:**
   - Переименовать `src/utils/tinkoff.ts` → `src/utils/tinkoff-production.ts` (сохранить боевую версию)
   - Переименовать `src/utils/tinkoff-demo.ts` → `src/utils/tinkoff.ts` (использовать DEMO)

2. **Задеплоить на Firebase:**
   ```bash
   npm run build
   firebase deploy --only hosting
   ```

---

## Тест №7: Формирование чека

### Шаг 1: Создать заказ
1. Зайти на https://asia-sib.web.app
2. Добавить любой товар в корзину
3. Оформить заказ
4. Нажать "Оплатить заказ"

### Шаг 2: Оплатить тестовой картой
На форме Тинькофф ввести:
- **Номер карты:** `4000 0000 0000 0101`
- **Срок:** `12/30`
- **CVV:** `111`
- Нажать "Оплатить"

### Шаг 3: Проверить
1. Зайти в личный кабинет Тинькофф Бизнес
2. Раздел "Операции"
3. Найти платеж - статус должен быть "Подтвержден"
4. Вернуться на страницу тестов
5. Тест №7 должен стать "Пройден" ✅

---

## Тест №8: Отмена платежа

### Шаг 1: Создать заказ
1. Снова зайти на https://asia-sib.web.app
2. Создать новый заказ
3. Нажать "Оплатить заказ"

### Шаг 2: Оплатить тестовой картой
На форме Тинькофф ввести:
- **Номер карты:** `5000 0000 0000 0108`
- **Срок:** `12/30`
- **CVV:** `111`
- Нажать "Оплатить"

### Шаг 3: Отменить платеж

1. **Перейти на страницу отмены:**
   - Открыть https://asia-sib.web.app/test-cancel
   - PaymentId подставится автоматически из последнего платежа

2. **Нажать кнопку "Отменить платеж"**

3. **Дождаться подтверждения:**
   - Должно появиться сообщение "✅ Платеж успешно отменен!"

### Шаг 4: Проверить
1. Зайти в личный кабинет Тинькофф Бизнес
2. Раздел "Операции"
3. Найти платеж - статус должен быть "Возвращен полностью"
4. Вернуться на страницу тестов
5. Тест №8 должен стать "Пройден" ✅

---

## После прохождения тестов

1. **Вернуть боевую версию:**
   - Удалить `src/utils/tinkoff.ts` (DEMO версия)
   - Переименовать `src/utils/tinkoff-production.ts` → `src/utils/tinkoff.ts`

2. **Задеплоить боевую версию:**
   ```bash
   npm run build
   firebase deploy --only hosting
   ```

3. **Настроить webhook в Тинькофф:**
   - URL: `https://rinok.vercel.app/api/tinkoff-webhook`

---

## Готово! 🎉

После прохождения тестов Тинькофф активирует ваш терминал для приема реальных платежей.
