# Контрольная точка: Прямая оплата через Тинькофф

**Дата:** 2024
**Коммит:** `f202f5b`

## Что сделано

1. **Убрано модальное окно PaymentModal** - больше не показывается окно с реквизитами
2. **Прямой переход на форму Тинькофф** - при нажатии "Оплатить заказ" сразу открывается страница оплаты
3. **Интеграция с Тинькофф API** - создается платеж через API с параметром `PayType: 'O'` (все способы оплаты)
4. **Оплата картой или СБП** - пользователь выбирает способ на странице Тинькофф
5. **Деньги на Тинькофф Бизнес** - все платежи поступают на бизнес-аккаунт

## Изменённые файлы

- `src/pages/CustomerDashboard.tsx` - убран PaymentModal, добавлен прямой вызов API
- `src/components/PaymentModal.tsx` - оставлен для других мест (если нужно)
- `src/utils/tinkoff.ts` - API для работы с Тинькофф Эквайринг
